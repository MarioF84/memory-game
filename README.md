# Memory Game

A 2-player memory card-matching game built with React, TypeScript, Tailwind CSS, and Zustand.

## 🚀 Live Demo

**[Play the game here → https://mariof84.github.io/memory-game/](https://mariof84.github.io/memory-game/)**

## How to Play

1. Two players take turns flipping cards.
2. If both flipped cards match, the player scores a point and gets another turn.
3. If the cards don't match, they are flipped back and the other player takes a turn.
4. The game ends when all pairs have been found. The player with the most matches wins!

## Tech Stack

- [React](https://react.dev/) – UI library
- [TypeScript](https://www.typescriptlang.org/) – type safety
- [Vite](https://vite.dev/) – build tool
- [Tailwind CSS](https://tailwindcss.com/) – styling
- [Zustand](https://github.com/pmndrs/zustand) – state management

## Local Development

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```
